## Detailed Version (Full Description)

```
PokeCache Arena is an educational web application designed to demonstrate Redis caching strategies in a real-world context. Built with Next.js 16, Supabase PostgreSQL, and Prisma ORM, this project showcases the cache-aside (lazy loading) pattern used by companies like Facebook and Twitter to scale read-heavy workloads.

Key Features:
- Real-time cache metrics dashboard showing hits, misses, and hit rates
- Visual toast notifications for cache operations (hits, misses, clears)
- 50 Pokemon entries with complete stats (HP, Attack, Defense, Special Attack, Special Defense, Speed)
```
